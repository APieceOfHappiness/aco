1. Выбрать и реализовать один из метаэвристических алгоритмов для решения задачи маршрутизации транспорта с ограничениями грузоподъемности.

2. Осуществить подбор параметров алгоритмов, определить критерий остановки.

3. Провести вычислительные эксперименты на тестовых задачах (все из Set E, Set F, Set M, Set P): http://vrp.atd-lab.inf.puc-rio.br/index.php/en/

4. Исследовать зависимость качества и скорости получения решения относительно размерности задачи и типа задачи (E, F, M, P).

5. Оформить результаты в виде презентации 5-7 слайдов и прикрепить ссылку на репозиторий с кодом реализации.

6. Оценка выставляется после публичного представления результатов на семинаре.

Допустимые диапазоны среднего отклонения полученных решений от оптимальных: не более 10% для оценок 8-10, не более 15% для оценок 6-7, не более 20% для оценок не выше 5, не более 25% для оценок не выше 4.
